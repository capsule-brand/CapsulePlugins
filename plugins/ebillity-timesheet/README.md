# eBillity Timesheet

Manages time in **eBillity TimeTracker** by driving Chrome through the Claude extension.

- **Log / edit / move** time by job + day + hours (resolves the service item for you)
- **Live timer** — start, pause, stop against a job
- **Answer** questions about recorded time ("how many hours on AMP-049 this week?")
- **Readiness check, submit, approve, fix rejected** — gated, only on explicit request

**Safety:** never types your password, and never submits, approves, or deletes on its own initiative — those require an explicit, named instruction. Logging leaves entries in **Pending**.

Designed in-house by Capsule.
